Special Element

Special Element

0.4

<content>

The element serves as content distribution outlest in a composed component template. The element itself will be replaced.

alias: <slot>

Code Example

<we-element name="item">
  <template>
    <div>
      <content></content>
    </div>
  </template>
</we-element>

<template>
  <div>
    <item>
      <text>Content Text</text>
    </item>
  </div>
</template>
